虚拟DOM  
并不是真实的DOM节点，而是存在于内存之中的一种数据结构。  

设计思想：虚拟DOM是用来找出怎么以最小的代价来更新DOM，保证最小化的DOM操作，使得执行效率得到保证  

步骤一：用JS对象模拟DOM树
步骤二：比较两棵虚拟DOM树的差异
步骤三：把差异应用到真正的DOM树上
